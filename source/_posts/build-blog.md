这篇博客就是用纯 HTML + CSS 搭建的。没有框架，没有构建工具，没有任何依赖。

## 项目结构

```
my-blog/
├── index.html          # 首页
├── posts/              # 文章目录
│   ├── hello-world.html
│   └── build-blog.html
├── css/
│   └── style.css       # 全局样式
└── nginx.conf          # Nginx 配置
```

## 为什么选择纯静态？

1. **极致的性能** —— 不需要任何运行时，Nginx 直接返回 HTML 文件。
2. **零维护成本** —— 没有依赖，没有版本升级，写完就忘。
3. **完全的控制权** —— 每一个像素、每一个间距都是自己决定的。
4. **部署简单** —— 把文件丢到服务器上就完事了。

最适合个人博客的，往往是最简单的方案。
